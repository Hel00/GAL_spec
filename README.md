# GAL — General Applicative Language

GAL is a systems programming language designed around compile-time metaprogramming, uniform calling conventions, and a compact but expressive syntax.

## Documents

- [SPECIFICATION.md](SPECIFICATION.md) — the full language specification
- [GRAMMAR.md](GRAMMAR.md) — consolidated BNF grammar reference
- [GLOSSARY.md](GLOSSARY.md) — definitions of all terms used in the specification
- [READING.md](READING.md) — how to read the specification
- [MEMORY_LAYOUT.md](MEMORY_LAYOUT.md) — binary layout rules for all types in terms of abstract size units
- [SEMANTICS.md](SEMANTICS.md) — procedural type checking and semantic analysis rules
- [STAGES.md](STAGES.md) — compiler pipeline stages mapped to spec sections and semantic rules
